## Getting Started

0. Setting Firebase
  Authentication, Firestore Database, Storage

### Installation
1. Clone this repo

```bash
git clone https://github.com/ofgarden/Cabinet_of_Curiosities.git 
cd Cabinet_of_Curiosities
``` 

2. Install dependencies

```bash
npm install
``` 

3. Start developmet server

```bash
expo start
```
